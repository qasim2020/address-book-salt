# Address-book-salt
CRUD java app 
- ``/findAll`` — Get all records
- ``/findById/{id}`` - Get a record by its id
- ``/createOne`` — Post a new record
- ``/deleteOne/{id}`` - Delete a record by its id
- ``/updateOne/{id}`` — Put (updates) a record by its id
